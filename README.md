# 🎵 GDStudio 音乐源 - 洛雪音乐助手自定义音源

## 📖 项目简介

这是一个为 **洛雪音乐助手（lx-music-desktop和lx-music-mobile）** 开发的自定义音源脚本，基于 GD音乐台（music.gdstudio.xyz）的 API 构建。支持网易云音乐、酷我音乐等多个平台的音乐播放，无需复杂配置，下载即用。

## ✨ 功能特性

- ✅ **多平台支持**：网易云音乐、酷我音乐
- ✅ **多音质支持**：128k、192k、320k、FLAC（自动降级）
- ✅ **简单易用**：下载脚本 → 导入软件 → 立即使用
- ✅ **详细日志**：便于调试和问题排查
- ✅ **自动更新**：脚本会根据 API 更新自动适配

## 🚀 快速使用

### 安装步骤（3步搞定）

1. **下载脚本**
   - 点击**洛雪音乐自定义音源-基于GD音乐台-v1.0.0.js**文件
   - 点击右上角 "Raw" 按钮
   - 右键 → 另存为，保存到本地

2. **导入洛雪音乐助手**
   - 打开洛雪音乐助手
   - 进入【设置】→【音乐来源】→【自定义源管理】
   - 点击【添加新源】，选择刚刚下载的 `洛雪音乐自定义音源-基于GD音乐台-v1.0.0.js` 文件

3. **开始使用**
   - 在左侧来源列表中选择 "统一音乐源"
   - 搜索音乐 → 播放 → 享受！

## 🎯 支持的音源

| 音源名称 | 源标识 | 支持音质 |
|----------|--------|----------|
| **酷我音乐** | `kw` | 128k, 192k, 320k, FLAC |
| **网易云音乐** | `wy` | 128k, 320k, FLAC， FLAC24bit |

## 📊 音质说明

脚本支持智能音质降级机制：
- 优先尝试最高音质（如 FLAC）
- 如不可用，自动降级到下一音质等级
- 保证始终有可播放的链接

## 🐛 常见问题

### Q: 导入后无法搜索音乐？
**A**: 检查以下事项：
1. 确保网络连接正常
2. 检查洛雪音乐助手是否为最新版本
3. 尝试重启洛雪音乐助手

### Q: 播放时提示"无法获取链接"？
**A**: 
1. 可能是 API 临时维护，请稍后重试
2. 检查是否达到 API 请求限制（5分钟50次）
3. 尝试切换到其他音源测试

### Q: 如何更新脚本？
**A**: 
1. 下载最新版本的脚本文件
2. 在洛雪音乐助手中删除旧源
3. 重新导入新版本的脚本

### Q: 支持其他音乐平台吗？
**A**: 目前仅支持列表中的平台。如需新增平台，请关注 API 提供者的更新。
